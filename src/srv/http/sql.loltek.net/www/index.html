sql.loltek.net

A pair of public MySQL and PostgreSQL servers for quick experiments, demos, and blog posts. They are intentionally disposable: no availability guarantees, and the data may be wiped at any time.

Credentials

  • MySQL: host sql.loltek.net, port 3306, database test, user test, password test
  • PostgreSQL: host sql.loltek.net, port 5432, database test, user test, password test

Admin UI: /adminer/ (Adminer)

CLI Examples

mysql -h sql.loltek.net -P 3306 -u test -ptest test
psql "host=sql.loltek.net port=5432 dbname=test user=test password=test sslmode=require"

PHP PDO (MySQL)

<?php
$db = new \PDO(
    "mysql:host=sql.loltek.net;dbname=test;charset=utf8mb4",
    'test',
    'test',
    [
        \PDO::ATTR_EMULATE_PREPARES => false,
        \PDO::ATTR_ERRMODE => \PDO::ERRMODE_EXCEPTION,
    ]
);
$stmt = $db->query("SELECT NOW() AS server_time");
echo $stmt->fetchColumn();

PHP PDO (PostgreSQL)

<?php
$db = new \PDO(
    "pgsql:host=sql.loltek.net;port=5432;dbname=test",
    'test',
    'test',
    [
        \PDO::ATTR_ERRMODE => \PDO::ERRMODE_EXCEPTION,
    ]
);
$stmt = $db->query("SELECT NOW() AS server_time");
echo $stmt->fetchColumn();

Please be considerate: these databases are shared by everyone. Avoid long-running jobs, remove any sensitive data, and remember that resets happen without notice.
